## Requirements
- Should have nodejs

###  Clone the Repo

`$ npm install`

###  Running  tests
`$ npx hardhat test`

### Hardhat node
`$ npx hardhat node`

### deploy open new teriminal
`$ npx hardhat run ./scripts/deploy.js --network localhost`

### Start frontend (open new terminal)
`$ npm run start`
